Navigation

The navigation compenent is used to for the navigation in your application.

Pds-App-Nav Component

<pds-app-nav class="pds-app-nav"
             app-name="My App Name"
             [navigation]="navigationObject"
             [user-navigation]="userNavigationObject"
             [user]="userObject">
</pds-app-nav>

The objects passed into the component are required to be these types: 

import {NavItem} from 'powerschool-design-system/core';

navigationObject: Array<NavItem>;

userNavigationObject: Array<NavItem>;

userObject: { fistName: string, lastName: string };

NavItem Object

The navigation and user navigation are built using arrays of NavItem objects. These objects have the following properties: 

id?: string; //the id attribute on the navigation link
name: string; //the content of the navigation link
iconClass?: string; //the class of the icon of the navigation link
hasSecondary?: boolean; // whether this navigation link has children that will be displayed in the secondary menu
activeAt?: string; // regular express that will be used to set the active class on a navigation link
route?: string; // the navigation component will use the Angular router when the navigation link is clicked
children?: Array<NavItem>; // a list of child navigation links to be displayed in the secondary menu
isOpen?: boolean; // is TRUE when the secondary menu of this navigation link is open
isActive?: boolean; // is TRUE when the navigation link is active
callback?: Function; // if a function is passed in, the navigation component will call this function instead of using the Angular router when the navigation link is clicked

An example of setting up the navigation array: 

let navigationObject: Array<NavItem> = [{
    id: 'first-main-nav-link',
    name: 'First Main',
    activeAt: '^/first-main',
    route: '/first-main',
    iconClass: 'icon-class'
}, {
    id: 'second-with-children-nav-link',
    name: 'Second With Children',
    hasSecondary: true,
    iconClass: 'icon-class',
    children: [{
        id: 'first-child-nav-link',
        name: 'First Child',
        activeAt: '^/second-main/first-child',
        route: '/second-main/first-child'
    },
    {
        id: 'second-child-nav-link',
        name: 'Second Child',
        activeAt: '^/second-main/second-child',
        route: '/second-main/second-child'
    }]
}, {
    id: 'third-main-nav-link',
    name: 'Third Main Callback',
    activeAt: '^/third-main',
    callback: this.myNavCallback.bind(this),
    iconClass: 'icon-class'
}];
